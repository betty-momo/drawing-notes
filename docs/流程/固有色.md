# 固有色

## 1.做灰底

具体操作：

（1）.将线稿图层勾选指定为选取样本

（2）.创建灰底图层用魔棒工具反选填充灰底颜色（线稿闭合不完全可以尝试反选或者选区笔闭合线稿，灰底颜色能和二分有明显对比就行，不需要仔细斟酌）

（3）.创建一个固有色图层，以灰底为基准创建一个剪贴蒙版并填充好自己想要的固有色（分部位处理，创建大致图层，比如头发，皮肤，衣服等等，根据熟练度不同来决定图层分细不细，学习的前期可以分的粗一点）

## 2.固有色的选择

选择低饱和，高明度的颜色进行填充

过程中可以创建一个饱和度图层来检查画面中的黑白灰关系

## 3.固有色阶段可以做一些特殊处理

例如人物头发部分，就可以靠近脸部就可以用喷枪刷淡橘黄色的暖色，远离的部分比如发尾可以用喷枪刷一些深蓝色的冷色让人物头发更加通透舒适
