# 灰面

笔刷：选区笔 + 喷枪

- **作用**：解决喷枪边缘难以控制的问题。先用选区笔确定边缘，再用喷枪上色，最后可用模糊工具适当柔化
- **适用**：非常适合塑造灰面

1.眼睫毛不要用实心，不太好看，SAI2用三根毛发笔完成完成，体会一根线包裹另一根线造成上翘的感觉，眼睫毛可以加入反光一类的东西塑造更强的体积感，下睫毛不要做太明显了，眼睫毛会透部分肤色表现出通透感

<img src="../img/眼睛.png" style="zoom: 25%;" />

2.眼球牢记上深下浅，同色系，最上层可以做一些深色杂乱处理以及最上方一部分可以用纯白色塑造出晶莹剔透的效果然后是俩边可以做一些排线丰富效果，以及画出虹膜增加体积感和丰富度，深色那边可以少量用其他颜色做些冷暖对比

3.瞳孔目前来看饱和度偏低，外圈明显
