# 高光

## 面部

添加高光

- 新建**发光（Glow）**图层。
- 先用喷枪轻喷鼻子高光区域及靠近光源的额头三角区。
- 使用**选区笔+喷枪**组合，在之前规划的高点（眉弓、颧骨等）点出高光。若无画颧骨习惯，可点在两边脸颊上。
- 下巴靠近光源处也可轻微带过，但不要太明显。

## 图层混合模式应用

- **提亮/提高饱和度**：使用 **叠加（Overlay）** 图层。
- **较弱的高光或亮面/反光**：使用 **强光（Hard Light）** 图层。
- **较强的高光**：使用 **发光（Glow）** 图层。
- **极端情况（如金属、太阳光直射）**：使用 **线性减淡（Linear Dodge）** 图层。
- **其他主观处理**：使用 **普通（Normal）** 图层即可
