# 三分

*   **腮红**:
    *   单独创建普通图层，反复调整位置和形状，不必一笔成形。
    *   线状的深色部分可以角度交错，避免呆板。
*   **弱结构阴影**:
    *   新建 **正片叠底** 图层，使用 **马克笔** 的半透明效果。
    *   刻画部位：双眼皮、眉弓骨、脸颊转折、鼻翼、下巴等。
        *   此步骤能产生细微层次变化，同时保持平面感，是二次元绘画的主要思路之一。

* **增强透感**:

    *   在双眼皮等处加几笔高饱和度的红色，增强皮肤透感，并与后续眼睛的高饱和上色呼应。
  *   **眼白与眼影**:
      *   **眼白**: 新建普通图层，可直接取纯白色，用半透明的马克笔涂一层，自然透出底下肤色。阴影可做冷暖变化，交界处加一笔红色，边缘用模糊工具过渡
  *   **嘴唇**:
      *   画法同眼影，不用太明显，避免显得嘴唇厚。
  *   **局部调整与重色阴影**:
      *   用马克笔补充细节，如下巴轮廓、颜色不够深的阴影等。
      *   新建 **正片叠底** 图层，加重色阴影（通常为闭塞阴影和投影）。
      *   例如在刘海中间用饱和度稍高的颜色，增加透光感。这一层边缘尽量画硬，与之前处理方法形成对比。
  *   **高光与提亮**:
      *   新建 **发光** 图层，用喷枪喷亮部。
      *   **高光** 需先用选区笔或套索工具确定位置、形状、大小，再用喷枪，最后用模糊工具过渡。要主动用橡皮修整形状。
  *   **肤色提亮与线稿染色**:
      *   流程中肤色易画黑画脏，可在二分图层上新建 **正片叠底** 图层，喷明度较高的暖色进行提亮，此操作后续可能重复。
      *   **线稿染色技巧**: 完成局部上色（如面部）后，切换到对应线稿图层，锁定透明像素，用喷枪吸取附近深色（如面部用高饱和红色）直接喷在线稿上。五官外轮廓受肤色影响大，可多喷一些。此方法能让线稿更融入画面。
