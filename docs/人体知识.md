# 人体知识

- 五官比例更偏好"三庭三眼"（而非三庭五眼）。可以用九宫格辅助定位。

  - **检查眼睛透视的方法**：沿头部透视画三条弧线——1) 眉弓到耳朵；2) 眼角连线（有时眼角眼尾需分开）；3) 眼睛底端连线。确保眼睛透视正确

- 头肩颈

  - 肩宽约为1.25个头宽
  - 注意透视：近镜头一侧肩膀略长，但需保持两侧肩宽基本等长
  - 收细脖子时，参考"下巴到锁骨（一动笔）"的线条

- 腰腹

  - 二次元的腰更短、更细
  - 需强调**腹外斜肌**和**盆骨骨点**等转折处，这些肌肉挤压的细节很出效果
  - 照片参考被遮挡的部分，需要额外找参考补充

- 腿部

  - 二次元的腿更细、稍短
  - 关键结构点：**缝匠肌**（在照片上明显）、**大转子**（下半身最宽处）

- 手臂与手

  - 手臂整体比照片更细、短
  - **检查臂长方法**：将手肘按在小腹上，手肘位置应比肚脐略高
  - 手画得比照片小，手掌约等于脸颊大小

## 区分男女以及年龄的关键

男性眼睛靠上，脖子更粗，肌肉更明显，身体曲线会弱化一部分，眉毛会更粗，肩宽会比1.5头宽一些，眼睫毛会弱化一部分（具体比例取决于男生类型）

女性眼睛靠下，脖子细一些，肌肉不明显，身体特别是腰部臀部带有明显曲线，盆骨更大，眉毛纤细，肩宽在1.5头及以下，眼睫毛明显（幼女肩宽更细）

## 五官定位与透视

*   **眼睛**：
    *   用四个折线代表睫毛，折线可暗示透视。
    *   用弧线标黑眼球位置，可适当扩大。
    *   **透视纠正**：人物头部透视更多是"折叠"关系，而非简单的近大远小。离镜头远的眼睛是**变窄**（高度不变），而不是变小。
*   **实用技巧——九宫格辅助法**：
    *   用变形工具给九宫格拉出面部透视。
        *   **定位参考**：
            *   最上端：发际线。
                *   最下端中间：下巴。
                *   左右中间：眼睛转折处。
                *   眉毛：放在最上面一列的中间，根据透视，近处眉毛可略长越过中线，远处略短。
                *   眼睛上睫毛：底下第二条横线处，同样遵循透视（略超过/正好/略短）。
                *   鼻子：第三条横线中间，二次元鼻尖略高于中心。
                *   嘴巴：位于最下面中间的格子。
                *   耳朵：比真人更集中，距离眼睛约半格宽度，顶端对应眼尾，底端对应下颌线。
