# 头发塑造笔记

## 核心概念：质感与二次元风格

*   **质感定义**：指描绘的物体（如玻璃、金属）看起来像其本身材质。提升质感的第一步是耐心区分每一层素描关系的外轮廓。
*   **二次元风格的特殊性**：二次元是现实简化后的风格化表达，尤其是头发。需要避免添加过多的现实质感，否则会显得"脏"或"乱"。
*   **核心原则**：画好整体二分的形状和位置，其重要性**远远大于**增加质感细节。对于新手，建议仅在每组头发的交接处添加少量细节，整体保持平整干净。

## 头发塑造步骤详解

### 第一步：整理与细化二分（阴影）

*   **目标**：让头发的阴影看起来更像头发，而不是简单的色块。

* **操作**：

    1.  在已有的整体二分图层上进行调整。
    2.  添加头发交错的阴影边缘，以增加层次感和质感。
        3.  二次元素描关系可分为：高光、反光、阴影三种。此步骤处理的是面积最大的阴影部分。

*   **关键提醒**：避免因过度添加交错阴影而导致画面"花"得像"狗啃的"。

### 第二步：添加底色与冷暖变化

* **操作**：
    1.  新建**正片叠底**图层，取头发的阴影色。
    2.  根据画面整体色调定位（如偏冷色调），在色相上微调（如往蓝色拉）。
    3.  将头部视为球形，用**喷枪**从下往上（背光面）喷一些过渡，增加立体感。
    4.  此图层应**降低存在感**，最好一眼看不太出来。
    5.  可在底色上新建普通图层，添加头发颜色的渐变。

### 第三步：塑造灰面（三分）

* **定义**：比二分颜色浅、外轮廓更偏向于表现质感的阴影层，可理解为"三分"。

* **操作与笔刷**：

    *   在二分图层上新建**正片叠底**图层。
    *   几乎全程使用**马克笔**，笔触模仿发丝的排线。
    *   绝大部分笔触是从头发末端向上"滑"出尖角。
    *   大面积区域：先顺着头发结构，画出"两头尖中间宽"的一笔，然后用**橡皮擦**擦出交错感。
    *   发束分岔处：从一端向另一端"扫"出分岔。

  *   **核心技巧与避坑**：
      1.  **避免杂乱**：小范围内不要画太多笔触。如果不小心画多了，可以用一个大笔触把多个小笔触合并。
      2.  **范围控制**：三分的阴影范围**不要越过二分**的边界（即不能一笔横跨亮面和暗面）。特殊情况可用大面积三分将二分包含在内。
        3.  **取色**：笔刷取色工具选"当前图层"，方便取当前图层的阴影色。
      4.  **下笔果断**：马克笔下笔尽量一笔成型，画错就撤销，不要反复涂抹，否则会显得"脏"。

  *   **不同部位的画法**：
      *   **刘海**：可能比较平，添加一些三角形阴影即可。
      *   **马尾/披散的后发**：意识地区分发组之间的前后、转折、遮挡关系。主要思路依然是沿着线稿结构添加"两头尖中间宽"的笔触。

  *   **关于灰面塑造的思考**：
      *   这是一个**比较偏"吃力不讨好"**的步骤。做得不好会显脏乱，做得不明显又难以被注意到，只有做到出色才会好看，但耗时可能远超其他部分。
      *   在追求**快速出图**或**平面感强**的作品（如部分二游、动画风格）中，可以弱化甚至省略此步骤，转而在图形外轮廓或花纹上下功夫。
      *   **建议**：要画就尽全力画好，得过且过反而会拉低整体观感。

### 第四步：塑造光源（亮部与高光/反光）

*   **4.1 塑造亮部（受光面）**：
    *   新建**发光图层**。
    *   按住 `Ctrl/Cmd` 单击二分图层缩略图获取选区，然后**反选**，得到亮面范围。
    *   选一个偏阳光的低饱和度暖黄色，用**喷枪**在亮面上轻轻喷几下，营造被照亮的效果。完成后取消选区 (`Ctrl/Cmd + D`)。

*   **4.2 添加补光与区分层次**：
    *   新建**发光图层**，用于画高光、反光和补充光效。
    *   先用暖色喷在明暗交接处，提高饱和度（常用且好出效果的上色方法）。
    *   对于结构不同但容易混在一起的部位（如马尾和鬓发），需要用明暗层次加以区分：
        *   **透光**：在背光但结构较薄处（如马尾边缘）补充小亮面。
        *   **压暗**：在受遮挡的部位（如手臂下的鬓发）加深阴影，形成"亮-暗-亮"的节奏对比。

*   **4.3 绘制反光（位于暗面）**：
    *   在头发背光面，与亮部的暖色形成对比，可以添加一些冷色反光（可取附近环境色，如蓝绿色）。
        *   反光是反射周围环境光，可以在颜色上添加更丰富的冷暖对比，这是背光效果好的重要原因。
        *   **绘制技巧**：
            1.  先用**选区笔**定位。
            2.  用**选区擦**擦出每一组头发反光的形状。
            3.  用**喷枪**稍微喷点亮色。
            4.  可用橡皮擦调整形状，用模糊工具做过渡。

*   **4.4 绘制高光（位于亮面）**：
    *   **位置**：高光位于头发的**鼓点**（突起的结构面），而非物理最高点。体现在头发上，是每一组头发偏中心靠上的位置。不要凭感觉乱加，否则会显"油腻"。
    *   **形状**：类似于梭型/菱形。与金属高光形状类似，区别在于需要添加纵向的、深色的发丝感来破型。
    *   **绘制技巧**：
        1.  同样先用**选区笔**和**选区擦**确定形状和位置。
        2.  直接用**喷枪**提亮（亮部高光颜色变化微妙，直接提亮即可）。
        3.  **数量要少**：不需要每组头发都加，只需在某个大块区域添加一个或连续一排即可。少量添加更容易出"画龙点睛"的效果。
    *   **风格化**：二次元绘画中高光形状可以很风格化，如云朵、星星形状，会更"二次元"。本视频画法偏写实一点。
    *   **特殊处理**：在刘海底端等靠近皮肤的位置，可以选偏皮肤的暖色喷一些，让头发显得更"透气"，但要记得擦掉被深色头发遮挡的部分。

### 第五步：环境光补充与整体调整

*   **环境光补充**：
    *   主观地在暗部添加带有明显颜色倾向的光斑，以丰富画面。
    *   **示例**：靠近皮肤处用暖色，靠近冷色天空处用冷色，或用紫色等中间色补充在大面积单调颜色处形成反差。
    *   **原则**：适量去做，可以明显但不要过多，主要目的是丰富画面，**不能破坏整体的干净和平整**。

*   **重要的检查习惯**：
    *   如果当前阶段感觉画面有问题，建议**返回检查上一个阶段**。
    *   **示例**：高光加完后感觉层次不够，就返回阴影部分去压暗。
    *   避免画得上头一直推进，最后发现问题时修改成本太高或不敢下笔。应在绘画过程中就经常返回检查。

### 第六步：线稿染色与最终细节

*   **线稿染色**：
    1.  切换到头发线稿图层，**锁定透明像素**。
    2.  选取附近较深的阴影色，用**喷枪**让线稿融入画面。
    3.  具体上色：刘海选深暖色，发尾选深蓝绿色，头发周围选头发阴影的暖色，偏亮的地方选光源的暖黄色。

*   **添加投影与透气感**：
    1.  在五官图层组新建图层，选**创建剪贴蒙版**。
    2.  取色工具选"所有图层"，用**硬边笔**选取五官附近头发的颜色，画出头发在皮肤上的投影，增加透气感。

*   **最终发丝细节**：
    *   在线稿图层最上方新建图层，用笔勾出一些分散的单独发丝。
    *   取色一般直接选头发亮部的颜色，这样更明显。
    *   总体目的是在头发轮廓比较单调、变化少的地方，添加一些细节装饰。

## 常见问题与心态建议

*   **不敢压暗浅色头发**：导致头发过平没有层次。可以尝试：先压暗比较好确定的闭塞和投影；或新建正片叠底图层，缩小画布，用大笔触马克笔去压暗阴影。
*   **发束层次画不好**：这是普遍难点，极度依赖长期的经验积累。即使对着参考画效果不好也很正常，不必自我怀疑。
*   **学习建议**：
    1.  **先养成思路**：在每次绘画中，有意识地去表现头发的参差、转折、遮挡关系。
    2.  **理性分析结构**：画画需要感觉，但不能只靠感觉，理性地分析、练习结构带来的效果可能更好。
    3.  **观察与总结套路**：大部分发型比较固定。练习时可多观察总结大佬的套路（如特定阴影形状出现的位置），下次画自己的图时尝试加上，久而久之形成自己的上色方法。

关于练习：塑造头发的灰面、高光/反光形状等步骤，需要大量的临摹或参考练习，初期很难把握是正常的，需要时间和耐心。

## 笔刷说明

### 马克笔

- **特性**：半透明圆头笔，适合做颜色倾向。
- **优点**：无需精确选色。例如画头发暗部时，可直接选一个低饱和度冷色涂上，半透明感能自然透出底色，避免生硬。
- **使用注意**：**一次性**使用。若一次涂不好，应整体撤销重画，避免反复涂抹导致笔触脏乱。

## 详细塑造手法

*   最常用的笔刷是**马克笔**。用于在暗部（二分阴影）上再画一层更暗的阴影，可以理解为"暗面的暗面"或"三分、四分"。

*   **绘制要点**：这一层的笔触要紧贴着暗面形状，不要同时画到暗面和亮面区域。
*   **检查修正**：如果笔触超出了暗面，要记得返回二分图层检查修正。
*   **目的差异**：二分更注重**图形感**，而这部分的塑造更强调**材质和结构**。

*   在头发部分，运用马克笔时，只需像"顺着头发线稿滑动"一样，用细的笔触带出发丝感即可。

*   **亮面处理**：大面积的亮面也需要用类似手法带出一些层次。
*   **发挥空间**：即使线稿将某片头发画得很大、很平面，在塑造时可以主观假设这里有"两股头发的交错"和偶尔形成的投影。
*   **审美与检查**：由于没有具体线稿规定，这部分非常依赖个人审美，需要反复**缩小画布检查**，避免画得"油腻"。形状和位置也需要学习参考。

*   掌握"二分虚化"和"马克笔塑造"这两种基础手法，对于头发材质的塑造基本就够用了。

*   **技巧**：在画阴影时，可以尽量画得"大"且"整"，然后新建图层去画更小、更细的地方。这样等于画了两个层次，画面更丰富。对于像刘海边缘，用"大笔触+细笔触发丝"很容易出效果。

*   在亮面，可以补充被二分忽略的灰面。

*   对于远离线稿、位于中心区域的大片刘海，主要按照**装饰性**的思路去营造头发的交错感。
*   牢记流程是"一层叠着一层"的，画当前层时尽量"干净利落"，不要反复涂抹弄乱层次。感觉画错了就立刻撤销重画。

*   头发塑造的难点在于需要注意的结构太多，非常考验耐心。

*   塑造后，可以"把笔刷调暗一些"，稍微强调一下交界线和投影，丰富细节。
*   注意这一步也起装饰作用，不要做得太多，让画面看起来不太"平板"即可。

*   切换到头部（后脑勺）的塑造。这一层的阴影要**够暗**，对比度要更强，包括明度对比和**饱和度对比**。

*   解释了低保和度阴影的运用场景：头部受光少、反光也弱，相比受光多的区域饱和度自然更低，因此需要降低饱和度。

*   结构的上下前后区分很大程度上依赖于投影。要在每一组头发对应的线稿处尽量压暗，并稍微做出头发的体积感。

*   完成上述两步后，头发的层次基本完整。可以新建一个**正片叠底**图层，取一个亮色，用喷枪轻轻做整体过渡，注意尽量清透，不要破坏已做好的素描关系。

*   在头发图层上新建一个"塑造图层"，用于处理混合模式解决不了的效果。
*   例如，取皮肤的红光，用喷枪在靠近脸部的地方做一点"透肉感"。对于灰白发效果不明显，如果是偏冷色的头发，这一步会很加分。

## 高光、反光与细节刻画

*   新建**强光**图层来画高光和反光。
*   因为整体偏俯视，头发高光会沿着"下弧线"分布。使用**选区笔**，用类似画暗面头发交界的手法，快速圈出形状，再用**选区擦**整理。
*   喷枪上色：靠近光源用**黄色**，远离的地方可以主观处理。

*   可以沿着色轮添加一些"五颜六色"的效果。反复套用"靠近光源用黄光"的规律来统一画面，减少违和感。

*   如果强光图层不够亮，可以在**发光图层**上再轻轻喷一些。
*   在暗部也可以画主观的装饰光：例如靠近脸部/服饰的地方，用细笔画点红色；在光源和服饰影响不到的地方，大胆加入冷色反光与背景呼应。

*   亮部的高光可以直接使用亮色，用细笔刻画。

*   与脸部打光原理相同，在**发光图层**上用喷枪对亮面稍作提亮，做出明暗渐变。
*   同时在**强光图层**上，用饱和度稍高的暖色做一些过渡。

*   绘画时要**不断检查并修改**前面的图层。例如画高光时，如果发现某处阴影不够暗，要立刻返回加深。不要怕麻烦。

## 线稿处理与收尾工作

*   处理线稿：**锁定线稿图层的不透明度**，直接用喷枪在线稿上改色。
*   方法：在受光区域喷浅色，在背光区域喷深色。

*   后面的马尾重复相同流程（暗面、高光、提亮）。因为是靠后的部分，可以不必画得那么细致，注意做出前后空间感即可。

*   由于是偏背光的角度，马尾和脸部的二分可能区分不明显。既然脸部整体已被压暗，可以在马尾处做"**透光提亮**"，形成"亮对暗，暗对亮"的对比，这比无脑压黑暗部更出效果。

*   在五官图层组上新建图层，用喷枪喷出头发压在五官上的阴影效果。
*   在线稿图层上新建图层，用亮色配合硬边笔，画出更细致的发丝作为点缀，注意不要太多太乱。

*   在头发的固有色上，喷一点流萤自带的"发光"效果。
*   回到最上面的图层组，新建一个**叠加图层**，用对应的过渡色（如蓝色）进行整体调整。
