# 泳装衣褶笔记

## 问题概述与核心难点

本次视频主题是刻画贴身泳装（以流萤的泳装为例）的衣褶。核心难点在于：当布料紧贴平缓的皮肤时，线稿多为曲线，缺乏明显转折。上色后形成大面积、平缓的"灰面"，难以找到下手刻画阴影和体积感的切入点。

## 具体刻画流程与技巧

*   以流萤泳装为例，进入塑造阶段。流程是先铺浅色部分，但需注意避免浅色压得过暗，需要不断调整。由于底色分层细致，调整过程不算太麻烦，且这种调整本身就是对画面的整体优化。

*   对于明显的挤压部分，可以直接用硬笔顺着线稿画出阴影形状，并强调边缘的深色。

*   对于非常平整、感觉"加不上阴影"的区域，需要主观处理。

*   **主观创造衣褶理论**：衣物穿在身上，除了受力产生的褶皱，还有很多自然产生的褶皱。基于此，可以主观设定某处"在之前某个时间受过力"，从而添加一道轻微的褶皱。
    *   **技巧**：对于这种主观添加的轻微褶皱，推荐使用 **选区 + 喷枪** 的组合，便于控制轻重。笔触特点是一头硬、另一头模糊。

*   如果想增加更多层次或生活化细节，可以在多处添加这种主观的轻微褶皱层次。

## 不同结构区域的细化方法

*   **有具体结构的花边部分**：可以直接理性分析起伏结构，在凹下去的地方添加阴影。阴影的形状需要寻找现实参考。

*   **对二分阴影进行多次提亮时的注意事项**：
    *   此操作会破坏素描关系，可以先不管，后续通过颜色调整。
    *   素描关系后续可用 **正片叠底** 图层进行压暗。
    *   阴影本身尽量画得暗一些、硬一些，在远离结构线的地方进行模糊处理。

*   **"防内线"的运用**：沿着边缘画的线可统称为"防内线"（如崩坏三角色肩膀的线条）。画出这种线除了增加细节，还能暗示结构，应尽量添加。

*   **花边阴影的优化**：花边部分的阴影很大程度上受线稿限制。如果感觉阴影加不上或不好看，需要**返回线稿层面进行优化**，然后分析凹陷处并对应添加阴影。

*   **细化时的即时修改**：放大画布观察细节时，遇到之前没注意到的不合理之处，要即时修改，不要拖延（例如感觉某处需加截面，就直接回到底色层添加）。

*   **纯平缓大面的难点**：没有花边或装饰、一大片平缓弧面的情况更难处理（类似球面灰面），作者表示之后可能另出视频说明。

*   **裙摆部分**：因为与腿部互动产生支撑受力，反而更容易添加阴影。
    *   **技巧**：可直接参考照片，在直接下垂的地方以及结构转折旁边添加阴影。
    *   **常见错误与建议**：避免不敢压重阴影。要么不画，要画就画得明显一些。如果参考照片难以把握，更建议参考他人成熟作品，在确定的位置画得具体、明显些，这比模糊一团的效果更好。

*   **另一种处理思路**：对于难处理的区域，可以选择"流白"（留白），不做处理也是一种选择，有时反而显得画面更干净。

## 整体调整与氛围营造

*   **布料花纹的单独展示**：不同颜色的花纹，其形成的阴影色也会不同。二次元中花纹颜色可以艳一些、饱和度高一些，只要与画面整体不冲突即可。

*   **丰富环境光与颜色变化**：
    1.  **提亮亮面**：继续提亮亮部区域。
    2.  **强化明暗交界线**：在明暗交界线处添加高饱和度的颜色（属于"乌闹套工色"，即冷暖对比色）。
    3.  **添加环境色**：
        *   靠近皮肤的高光，可添加比较亮的肤色进行点缀。
        *   远离皮肤的地方，可添加一些背景色，但要尽量弱化其存在感，否则会使画面显乱。
        *   **一致性原则**：因为头发等处已经应用了这套添加环境色的"公式"，如果不在衣服上加，反而会显得头发部分很乱。
    4.  **透肤感处理**：贴近皮肤的布料，可以用喷枪喷上一些皮肤的颜色，以显得透肤。在有冷暖对比时，效果更佳。

## 总结与后续计划

*   **当前不足**：作者认为目前整体画面的明暗对比仍不够明显，空间感稍弱，已记下作为后期调整重点。
*   **后续计划**：主线部分（泳装刻画）大约还剩两期视频。此外，后期还需要补充皮肤的塑造部分。
